# Overview
- 支持 1320 种植物的识别.
- 模型大小为 51.8M, 准确率在 90% 以上.
- 模型开源, 易用的示例代码.
- 支持的植物物种以及识别准确率会持续更新.

# Dependency
- opencv-python 4.4.0
- flask

# Usage
Run `python main.py`, open `http://127.0.0.1:5000/`, and upload an image file.
